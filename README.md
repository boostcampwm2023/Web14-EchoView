 <div style="align-items : center">곰터뷰</div>
 
> 곰터뷰는 `무료`로 `로그인 없이` 사용할 수 있는 면접 연습 서비스입니다.  
> `프레임워크` 없이 `최소한의 라이브러리`로 웹 개발에 대한 `기초`를 다지려 노력했습니다.  
> `지속가능한` 프로젝트를 목표로 NCP가 아닌 `제한적인 환경`에서 서비스를 구축하기 위해 다양한 시도를 했으며,  
>  저희 팀원들이 많은 `성장`을 거듭할 수 있었던 프로젝트 입니다.
>
> 자랑하고 싶어서 매주 데모데이만를 기다렸습니다!  
> 우리의 프로젝트 `곰터뷰` 함께 구경해주시죠!!  
> https://www.gomterview.com
